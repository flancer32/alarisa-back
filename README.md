# @flancer32/alarisa-back

Main reusable server-side package for the Alarisa `back` area.

The package publishes TeqFW modules under the `Alarisa_Back_` namespace. It is not the self-hosted server composition root: that role remains with `@flancer32/alarisa`. Shared communication contracts belong to `@flancer32/alarisa-comm`.

The embedded cognitive context is in `ctx/`.
