# Product Overview

- Path: `ctx/docs/product/overview.md`
- Changed: `20260715`

## Product Identity

`@flancer32/alarisa-back` is the main reusable package for server-side Alarisa responsibilities.

## Product Scope

The accepted area includes orchestration, authorization and access control, durable state and Context management, agent and worker execution, scheduled and event-driven processing, and server-side integrations.

## Product Boundaries

The package is not the self-hosted server composition root, does not own shared server/client communication contracts, and does not own browser or native presentation.

## Current Boundary

The initial repository establishes package identity and ownership only. Concrete capabilities and modules require upstream product and architecture decisions.
