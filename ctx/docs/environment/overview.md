## Runtime Model

The package targets Node.js 20 or newer and is loaded as ESM into the self-hosted Alarisa server runtime. It does not define an independently deployed service.

## Environment Constraints

Concrete persistence, queue, scheduler, worker isolation, credential, and deployment technologies remain owned by upstream architecture and environment decisions.
